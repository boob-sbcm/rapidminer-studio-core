44    * This is an abstract class for all {@link AggregationFunction}s, that can be selected to aggregate
45    * values of a certain group. Each {@link AggregationFunction} must be able to provide a certain
46    * {@link Aggregator}, that will count the examples of one single group and compute the aggregated
